# djangobasicsGB

Учебный проект системы управления обучением, который разработан при прохождении обучения на факультете "Python-разработка", портал GB, курс "Основы Django".

## Стек

- Python > 3.7
  - isort, black, authoflake
  - Django < 3.3
  - Celery[Redis]
- VSCode
- SQLite 3
